### 405 Method Not Allowed (wrong HTTP verb)

Most of these are normal “you called an endpoint with a verb it doesn’t support”, e.g. trying PUT/PATCH/DELETE on endpoints that only implement GET/POST.

### 404 Not Found (wrong id/slug)

These are usually caused by calling detail endpoints with an id/slug that does not exist in the database.
